# Contributing to Wetware Engineering

感谢你对湿件工程项目的兴趣！

## 如何贡献

### 文档改进
- 修正错误或不清晰的表述
- 翻译文档到其他语言
- 添加示例和用例

### 规范建议
- 在 Issues 中提出 Bio-Component Spec 改进建议
- 讨论 Bio-DSL 语法扩展
- 提出新的接口类型定义

### 代码贡献
- 工具链开发
- 参考实现
- 测试框架

## 提交流程

1. Fork 本仓库
2. 创建特性分支 (`git checkout -b feature/your-feature`)
3. 提交更改 (`git commit -m 'Add some feature'`)
4. 推送到分支 (`git push origin feature/your-feature`)
5. 创建 Pull Request
