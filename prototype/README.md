# 湿件工程模拟器原型

## 快速运行

```bash
cd prototype
python wetware_sim.py
```

## 文件说明

| 文件 | 说明 |
|------|------|
| `wetware_sim.py` | 核心模拟器，包含三大模块和运行时 |

## 架构

```
┌─────────────────────────────────────────────────────────┐
│                    Bio-DSL 解析器                        │
│              (声明式系统描述 → 运行时配置)                 │
└─────────────────────────────────────────────────────────┘
                           ↓
┌─────────────────────────────────────────────────────────┐
│                    Bio-Runtime                          │
│         (组件注册 / 连接管理 / 信号传播 / 状态监控)        │
└─────────────────────────────────────────────────────────┘
                           ↓
┌─────────────┬─────────────┬─────────────────────────────┐
│ PowerModule │ Metabolism  │      ComputeModule          │
│   供能模块   │  代谢模块    │       计算决策模块           │
├─────────────┼─────────────┼─────────────────────────────┤
│ glucose_in  │ waste_in    │ sensor_in                   │
│ oxygen_in   │ co2_in      │ status_in                   │
│ atp_out     │ atp_in      │ atp_in                      │
│ co2_out     │ recycled_out│ control_out                 │
│ heat_out    │ status_out  │ decision_out                │
└─────────────┴─────────────┴─────────────────────────────┘
```

## Bio-DSL 语法

```biodsl
// 定义组件
COMPONENT <名称> FROM <类型>

// 连接端口
CONNECT <组件>.<端口> TO <组件>.<端口>

// 设置输入值
SET <组件>.<端口> = <数值>
```

## 扩展方向

1. **添加更多组件类型**：传感器、执行器、存储模块
2. **可视化**：用 matplotlib 或 Web 界面展示实时状态
3. **持久化**：将组件定义存入数据库（Bio-Registry）
4. **对接真实仿真器**：CompuCell3D、Tellurium
